Install the FLBoard FLHook Plugin
=================================
1. Get FLHook 2.0 plugin version.

2. Copy all DLL files in EXE folder in Freelancer's EXE folder. Those file are support lib.

3. Copy the board.dll and board.cfg file under EXE\flhook_plugins

4. Install other plugins that uses FLBoard.
